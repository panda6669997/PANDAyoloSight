## 2. 产品概述

### 2.1 产品定位

PADNAyoloSight 是一款基于 YOLOv8 深度学习模型的桌面端图片物体识别软件。用户可通过图形界面打开任意图片，一键运行目标检测，系统自动以彩色边界框标注识别到的物体，并提供类别名称、置信度等详细信息。

### 2.2 目标用户

- 需要快速识别图片中物体类别和位置的普通用户
- 计算机视觉学习者、研究人员
- 需要批量查看检测结果的测试人员

### 2.3 技术路线

```
Java 17 (Swing + FlatLaf GUI)
    → DJL (Deep Java Library) — Java 深度学习框架
    → ONNX Runtime — 跨平台推理引擎
    → YOLOv8n — 预训练目标检测模型（COCO 80类）
```

## 5. 系统架构

```
MainFrame (JFrame)
├── JMenuBar + JToolBar (打开/检测/保存/适应)
├── JSplitPane
│   ├── LEFT: ControlPanel (置信度/类别筛选)
│   └── RIGHT: JSplitPane
│       ├── TOP: JScrollPane → ImageCanvas (图片+框+缩放)
│       └── BOTTOM: ResultTablePanel (结果表格)
└── StatusBar

数据流:
  File → ImageUtils → BufferedImage
  BufferedImage → YoloDetector → List<DetectionResult>
  DetectionResult → ImageCanvas (绘制) + ResultTablePanel (表格)
  BufferedImage + DetectionResult → DrawingUtils → 保存图片
```
